# Cookbooks

Task-oriented, runnable recipes for this project. Each cookbook is a self-contained folder with a README and a runnable notebook.

- [Browser Harness Cookbook: CDP Automation with LLM Agents](./browser-harness-cookbook/README.md): Official cookbook for Browser Harness: connect LLMs to Chrome via CDP, write self-improving automation code, and build agent-driven browser tasks.
- [Manaflow Automation Cookbook: Orchestrate Parallel AI Agents](./manaflow-automation-cookbook/README.md): Automate development tasks with Manaflow. Spawn Claude Code, Codex CLI, and other AI agents in parallel isolated workspaces with automated PR creation.
- [Next.js + Browser Harness: Automate Testing & Scraping](./nextjs-browser-harness-automation/README.md): Build Next.js automation with Browser Harness: connect LLMs to Chrome via CDP, scrape pages, test apps, and write self-improving browser helpers in Python.
